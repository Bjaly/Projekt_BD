# Projekt Baza Danych Hurtownia Fitness
Baza danych zawiera dane sprzedażowe przykładowej hurtowni fitness

### Wykorzystane narzędzia:
  * MySQL 8.0
  * Hosting - Amazon Web Services RDS
  * RDMS - DBeaver    
  * Generowanie danych - ChatGPT4o
  * Raporty - Tablau
